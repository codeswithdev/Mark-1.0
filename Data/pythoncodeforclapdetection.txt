Here is a simple Python code that uses the `pyaudio` library to detect claps:

```python
import pyaudio
import numpy as np

# Set up audio parameters
FORMAT = pyaudio.paInt16
CHANNELS = 1
RATE = 44100
CHUNK = 1024

# Set up the audio threshold for clap detection
THRESHOLD = 2000

# Create a PyAudio object
p = pyaudio.PyAudio()

# Open the audio stream
stream = p.open(format=FORMAT, channels=CHANNELS, rate=RATE, input=True, frames_per_buffer=CHUNK)

print("Clap detection started. Press Ctrl+C to stop.")

try:
    while True:
        # Read audio data from the stream
        data = np.frombuffer(stream.read(CHUNK), dtype=np.int16)

        # Calculate the average amplitude of the audio data
        amplitude = np.average(np.abs(data))

        # Check if the amplitude is above the threshold
        if amplitude > THRESHOLD:
            print("Clap detected!")

except KeyboardInterrupt:
    # Close the audio stream and PyAudio object
    stream.stop_stream()
    stream.close()
    p.terminate()

    print("Clap detection stopped.")
```

This code works by continuously reading audio data from the default input device (usually the microphone) and calculating the average amplitude of the data. If the amplitude is above a certain threshold, it prints "Clap detected!".

Note: You will need to install the `pyaudio` library before running this code. You can install it using pip:

```bash
pip install pyaudio
```

Also, you may need to adjust the `THRESHOLD` value depending on the sensitivity of your microphone and the level of background noise.

Here's a more advanced version that uses a machine learning approach to detect claps:

```python
import pyaudio
import numpy as np
from sklearn.svm import OneClassSVM

# Set up audio parameters
FORMAT = pyaudio.paInt16
CHANNELS = 1
RATE = 44100
CHUNK = 1024

# Set up the audio threshold for clap detection
THRESHOLD = 2000

# Create a PyAudio object
p = pyaudio.PyAudio()

# Open the audio stream
stream = p.open(format=FORMAT, channels=CHANNELS, rate=RATE, input=True, frames_per_buffer=CHUNK)

# Create an SVM model for one-class classification
svm = OneClassSVM(kernel='rbf', gamma=0.1, nu=0.1)

# Train the model on background noise
print("Training model on background noise. Please be silent for 5 seconds.")
background_noise = []
for _ in range(5 * int(RATE / CHUNK)):
    data = np.frombuffer(stream.read(CHUNK), dtype=np.int16)
    background_noise.append(np.average(np.abs(data)))
svm.fit(np.array(background_noise).reshape(-1, 1))

print("Clap detection started. Press Ctrl+C to stop.")

try:
    while True:
        # Read audio data from the stream
        data = np.frombuffer(stream.read(CHUNK), dtype=np.int16)

        # Calculate the average amplitude of the audio data
        amplitude = np.average(np.abs(data))

        # Use the SVM model to predict if the audio data is a clap
        prediction = svm.predict(np.array([amplitude]).reshape(1, -1))

        # Check if the prediction is -1 (i.e., a clap)
        if prediction == -1:
            print("Clap detected!")

except KeyboardInterrupt:
    # Close the audio stream and PyAudio object
    stream.stop_stream()
    stream.close()
    p.terminate()

    print("Clap detection stopped.")
```

This code uses a one-class SVM model to classify audio data as either background noise or a clap. It first trains the model on background noise, then uses the model to predict if new audio data is a clap.
